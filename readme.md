### sloc ：区块链代码

### web ： 前端代码

### 版本：
Truffle : 5.11.5 (core: 5.11.5)  
Ganache : 7.9.1   
Solidity: 0.5.16 (solc-js)   
Node    : 16.20.2  
Web3.js : 1.10.0  
npm     : 8.19.4

### 运行：
1. 安装yarn：npm install yarn
2. 安装依赖：yarn
3. 部署合约：输入：truffle compile 输入：truffle migrate
3. 启动ganache
4. 启动前端：yarn serve
